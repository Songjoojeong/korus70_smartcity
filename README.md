# KORUS70 Smart City — 올랜도 도시권 교통–환경 복합 취약지역 분석

**Machine Learning-Based Analysis of Traffic–Air Quality Vulnerability in the Orlando Metropolitan Area**

> 한·미 첨단분야 청년교류 지원사업 (산업통상자원부) — 첨단산업 조사과제
> 주제: AI·데이터사이언스 기반 스마트시티 설계

미국 올랜도 도시권의 다출처 도시 데이터(대기질·교통·기상·대중교통·사회경제)를 결합하여 ① 일별 대기질을 예측하고, ② 교통 부담과 사회적 취약성을 통합한 복합 취약지역을 진단하며, ③ 정책 시나리오를 시뮬레이션한 학부 연구 프로젝트입니다.

---

## 1. 핵심 결과 요약

| 구분 | 내용 |
|------|------|
| **대기질 예측** | 교통 변수 추가 시 PM2.5 예측 성능 향상 (R² 0.07 → 0.41, RandomForest) |
| **SHAP 해석** | PM2.5 향상분 상당수는 전일 농도(자기상관)에 기인하며 교통 변수의 순기여는 제한적. 반면 NO2는 교통 거리·교통량이 SHAP 상위 → 교통–대기질 직접 연결은 NO2에서 더 명확 |
| **복합 취약성** | 교통량이 가장 많은 지점과 가장 취약한 지점이 **불일치** — 취약성은 사회경제적 요인에 더 크게 좌우 |
| **정책 시나리오** | 동일 예산이라도 형평 우선 배분이 최취약 집단을 더 직접 보호 / 취약도 ≠ 노출 인구 |

> 정책 시나리오는 복합 취약지수를 재계산하는 **시뮬레이션**이며, 실제 대기질 개선 효과의 정량적 예측이 아닙니다.

## 3. 분석 방법

데이터의 가용 기간 제약을 정면 대응하기 위해 **2-트랙 설계**를 채택했습니다.

- **트랙 A** (2023–2026, 교통 변수 제외): 장기간 대기질 모델
- **트랙 B** (2025–2026, 교통 변수 포함): 교통–환경 결합 모델

핵심 원칙:

1. **단일 좌표계 결합** — 모든 공간 데이터를 UTM 17N으로 정렬
2. **시계열 누수 방지** — 날짜 기준 앞 80% 학습 / 뒤 20% 검증 (무작위 분할 금지)
3. **모델 비교** — Baseline(mean) · Ridge · RandomForest · XGBoost, 평가지표 RMSE·MAE·R²
4. **이중 해석** — 순열 중요도(permutation importance)와 **SHAP**로 변수 기여를 교차검증
5. **복합 취약성·시나리오** — 예측 대기질·교통 부담·사회적 취약성을 가중 결합, 4종 정책 시나리오를 weighted overlay로 시뮬레이션

## 5. 실행 방법

Google Colab 환경을 기준으로 작성되었습니다.

```bash
# 주요 의존성
pip install geopandas scikit-learn xgboost shap matplotlib pandas
```

1. 위 데이터 출처에서 원자료를 내려받아 `data/` 경로에 배치
2. `korus70_smartcity.ipynb`를 Colab 또는 Jupyter에서 위에서부터 순서대로 실행
3. 결과 표·그림은 `outputs/`에 자동 저장

> 노트북은 셀 번호(`# %% [1] ~ [19]`)로 단계가 구분되어 있어, 데이터 로드 → 전처리 → 공간 결합 → 모델링(2-트랙) → 해석(순열·SHAP) → 취약성·시나리오 → 시각화 순으로 따라갈 수 있습니다.

## 6. 한계 및 유의사항

본 연구는 학부 수준의 사례 연구로, 다음 한계를 명시합니다.

- **소표본** — 분석 기간이 짧아(특히 트랙 B) 자기상관이 강하게 잡히며, 이로 인해 PM2.5 단기 예측은 전일 농도가 지배적입니다.
- **관측소 부족** — EPA 대기질 관측소가 적어 공간 일반화에 제약이 있습니다.
- **시뮬레이션 ≠ 인과** — 정책 시나리오는 취약지수 재계산이며 인과적 효과 추정이 아닙니다.
- **지역 특수성** — 올랜도 사례의 결과 수치를 다른 지역에 그대로 적용할 수 없으며, 본 저장소의 가치는 *재현 가능한 분석 절차*에 있습니다.
